# ⚡ NEON TANKS

بازی تانک نئونی روگ‌لایک — تک‌فایلی، بدون هیچ وابستگی، قابل اجرا در مرورگر و به‌صورت **Telegram Mini App** با پشتیبانی چندنفره (اسکواد).

**🎮 اجرا:** فایل [`tank.html`](tank.html) را در مرورگر باز کنید — همین!

## امکانات

- روگ‌لایک عمق‌محور با فروشگاه، گجت‌ها و پهپادها
- ۵ نوع باس + ۳ نوع مینی‌باس با الگوهای حمله‌ی متفاوت (خمپاره‌ی تلگراف‌شده، تلپورت، رگبار حلقه‌ای)
- کنترل لمسی کامل برای موبایل (دو جوی‌استیک شناور + دکمه‌های گجت)
- ادغام با Telegram Mini App (اکانت بازیکن = اکانت تلگرام)
- اسکواد تا ۴ نفره با کد دعوت و لینک `t.me` + پاداش‌های تیمی

## استقرار (Deploy) با Cloudflare — بدون سرور

این ریپو مستقیماً به Cloudflare وصل می‌شود:

1. **بازی (Cloudflare Pages):** در داشبورد کلادفلر → *Workers & Pages* → *Create* → *Pages* → *Connect to Git* → این ریپو را انتخاب کنید. بدون build command؛ خروجی همان ریشه‌ی ریپو است. آدرس HTTPS می‌گیرید.
2. **سرور چندنفره (اختیاری):** طبق راهنمای [`server/README.md`](server/README.md) با `wrangler deploy` روی Cloudflare Workers (پلن رایگان) مستقر کنید و سه متاتگ `mp-server` / `mp-bot` / `mp-app` را در `tank.html` تنظیم کنید.
3. **Telegram Mini App:** در BotFather با `/newbot` و `/newapp` ربات و مینی‌اپ بسازید و URL بازی را بدهید.

بدون سرور چندنفره، بازی کاملاً تک‌نفره و بدون خطا کار می‌کند.
